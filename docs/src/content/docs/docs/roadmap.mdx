### Release 2.0.0: Full Feature Set

- Link: [Milestone](https://github.com/preamza02/supakone.me/milestone/2)
- Expected Release Date: 7 December 2025

#### Functional Requirements

1. Personal Info Page Enhancements

- [ ] Sync Data across multiple platform (Github, Linkedin) Update Automatically
- [ ] Show My Activity Feed (Github Activity, Twitter Activity, Instagram Activity)
- [ ] Automated Resume Generation (PDF) using data from Personal Info Page
- [ ] Add Skills and Technologies section with visual proficiency indicators
- [ ] Add Timeline visualization for career and education milestones

2. Blog System Enhancements

- [ ] Have Share button for social media (Twitter/X, LinkedIn, Facebook)
- [ ] Add Reading time estimation for each blog post
- [ ] Add Related posts section at the end of each blog
- [ ] Implement Table of Contents auto-generation for long posts

3. Documentation Enhancements

- [ ] Show performance metrics for each project
- [ ] Add project status badges (active, maintained, deprecated)
- [ ] Implement version selector for documentation pages

#### Non-functional requirement

1. Performance

- [ ] Overall performance optimization (Lighthouse Performance Score > 90)
- [ ] Fast loading time less than 500 ms globally
- [ ] Show usage statistics for each project (Download count, Star count, Fork count)
- [ ] Implement image lazy loading with placeholder fallbacks
- [ ] Enable browser caching for static assets

2. Accessibility

- [ ] Ensure website is accessible to all users (Lighthouse Accessibility Score > 90)
- [ ] Add proper ARIA labels for interactive elements
- [ ] Ensure proper color contrast ratios for all text

3. Offline Support

- [ ] Implement PWA features for offline access
- [ ] Add service worker for caching essential pages
- [ ] Enable "Add to Home Screen" functionality

4. Security

- [ ] Ensure website is secure from common vulnerabilities (Lighthouse Best Practices Score > 90)
- [ ] Implement Content Security Policy (CSP) headers
- [ ] Enable HTTPS strict transport security

5. SEO

- [x] SEO optimized (Lighthouse SEO Score > 90)
- [ ] Add Open Graph meta tags for social media sharing
- [ ] Implement structured data (JSON-LD) for rich search results

6. Software Reliability

- [ ] Support All End-to-End Testing to ensure Software Reliability
- [ ] Add visual regression testing for UI components
- [ ] Implement automated accessibility testing in CI/CD pipeline
